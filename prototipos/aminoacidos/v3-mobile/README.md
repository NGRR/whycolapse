# Aminoácidos adaptativos · V3 móvil

Versión orientada a prueba en pantallas móviles.

## Cambios principales

- Recalcula radios orbitales según el tamaño real del `stage`.
- Los aminoácidos quedan dentro de una zona alcanzable en móvil.
- El lienzo puede desplazarse arrastrando una zona vacía del campo.
- Los nodos usan `touch-action: none` para evitar conflicto con gestos del navegador.
- Mantiene la memoria adaptativa con puntos naranjos interactivos.

## Uso

1. Abrir `index.html`.
2. Pulsar `INICIAR RETO`.
3. Arrastrar aminoácidos hacia eventos.
4. En móvil, arrastrar el fondo para mover el lienzo.
5. Al finalizar, tocar puntos naranjos para leer la memoria del ciclo.
